## 📝 第二步：AI 剧本创作

### 使用 Claude 开发剧本

**提示词模板：**
```
我想制作一部 [时长] 分钟的 AI 短片，类型是 [类型]。

我的初步想法是：[一句话概念]

请帮我开发一个一页纸的故事大纲，包括：
1. 核心冲突或问题
2. 场景设定和时间背景
3. 主角及其目标
4. 情感弧线

注意：保持"可通过 AI 生成镜头实现"，避免需要大量对话或特写镜头的场景。
```

**剧本优化要点：**
```
✅ 适合 AI 的故事类型：
- 视觉隐喻、氛围驱动
- 环境镜头、物体特写
- 剪影、远景镜头
- 少量角色、简单动作

❌ 避免：
- 大量对话场景
- 复杂角色互动
- 精确的面部表情
- 快速动作序列
```

## 🎬 第四部分：专业镜头提示词

### 提示词结构公式

```
[主体] + [动作] + [场景] + [光线] + [镜头运动和构图] + [风格/美学关键词]
```

### 完整提示词示例

**示例 1：开场镜头**
```
一位 60 多岁的老人穿着深色羊毛大衣独自坐在公园长椅上喂鸽子，
灰色阴天的早晨光线，中景镜头，静态相机，浅景深，
轻微去饱和的胶片颗粒，忧郁氛围，电影感 2.39:1 画幅，
35mm 镜头，柔和对比度，无文字叠加，无水印
```

**示例 2：动作镜头**
```
一个穿着红色风衣的女人在雨中奔跑穿过鹅卵石街道，
黄昏时分街灯刚刚亮起，水面反射橙色光芒，
跟踪镜头跟随她的运动，手持相机轻微晃动，
新黑色电影风格，高对比度，冷色调，电影质感
```

**示例 3：情绪镜头**
```
特写镜头：一双布满皱纹的手轻轻抚摸旧照片，
温暖的室内光线从窗户斜射进来，灰尘在光线中飘浮，
静态相机，极浅景深，背景虚化，
怀旧氛围，柔和色调，35mm 胶片质感，情感细腻
```

---

### 镜头术语词典

**镜头类型：**
```
- 大远景 (Extreme Wide Shot): 展示环境全貌
- 远景 (Wide Shot): 人物全身 + 环境
- 中景 (Medium Shot): 人物腰部以上
- 近景 (Close-up): 人物面部
- 特写 (Extreme Close-up): 细节特写（手、眼睛等）
```

**相机运动：**
```
- 推镜头 (Dolly In): 相机向前移动
- 拉镜头 (Dolly Out): 相机向后移动
- 摇镜头 (Pan): 相机水平旋转
- 移镜头 (Tracking Shot): 相机跟随主体移动
- 升降镜头 (Crane Shot): 相机垂直移动
- 手持 (Handheld): 轻微晃动，真实感
- 静态 (Static): 固定相机
```

**光线术语：**
```
- 黄金时刻 (Golden Hour): 日出/日落时的温暖光线
- 蓝色时刻 (Blue Hour): 日出前/日落后的冷色调
- 逆光 (Backlight): 光源在主体后方
- 侧光 (Side Light): 光源在侧面
- 柔光 (Soft Light): 漫射光，柔和阴影
- 硬光 (Hard Light): 直射光，强烈阴影
```

## 🎥 第五部分：场景一致性管理

### 角色一致性挑战

**问题：** AI 视频最难的是保持角色在不同镜头中外观一致

**解决方案：**

**方法 1: 参考图像法**
```
1. 花 10-15 次生成找到完美的角色外观
2. 保存最佳结果作为参考图像
3. 后续生成时上传此参考图像
4. 在提示词中描述："与参考图像相同的角色"
```

**方法 2: 减少角色曝光**
```
导演策略：
- 多用环境镜头
- 使用剪影和背影
- 远景代替近景
- 物体特写代替人物特写

这是合法的导演选择，不是技术妥协
```

**方法 3: 接受变化作为美学**
```
如果故事有超现实/心理元素，
角色的轻微变化可以成为风格特色
```

---

### 场景一致性检查清单

```
□ 色彩分级一致（使用相同 LUT/滤镜描述）
□ 时间/天气一致（避免同一场景不同时间）
□ 角色服装一致（详细描述服装）
□ 场景道具一致（关键道具在提示词中重复）
□ 光线方向一致（注意光源位置）
□ 画幅比例一致（全程使用相同比例）
```

## 📊 第六部分：拍摄比例管理

### 理解拍摄比例

**关键数字：**
```
预期拍摄比例：3:1 到 5:1

意味着：
- 每个可用镜头需要生成 3-5 次尝试
- 3 分钟电影需要 25-35 个镜头
- 总生成次数：75-175 次
```

**预算规划：**
```
以 Runway Gen-4 为例：
- 10 秒 1080p 镜头 ≈ $0.80-1.20
- 100 次生成 ≈ $80-120
- 3 分钟电影总视频成本：$150-300
```

---

### 失败镜头的常见原因

```
❌ 不自然的肢体运动或视觉伪影
❌ 角色外观漂移（不同脸型、服装细节）
❌ 运动模糊遮挡主体
❌ 背景与风格圣经冲突
❌ 动作太快或太慢不适合剪辑
❌ 物理效果不真实（水、重力等）
```

**应对策略：**
```
1. 保留所有生成，包括失败的
2. 失败的镜头可能适合其他场景
3. 建立"备用镜头库"
4. 为关键镜头预留更多预算
```

## 🎞️ 第七部分：剪辑原则

### AI 素材剪辑特殊技巧

**原则 1: 在运动上剪辑**
```
当出镜和入镜镜头都有视觉运动时，
转场感觉更流畅

静态到静态的剪辑对 AI 素材来说通常感觉突兀
```

**原则 2: 使用长镜头**
```
如果 AI 生成一个很棒的 8 秒镜头，
使用其中的 7 秒

AI 素材受益于 extended holds
```

**原则 3: 先调色**
```
在最终剪辑前统一调色
确保所有镜头色彩一致
```

---

### 剪辑流程

```
第 1 步：组装剪辑
- 按故事顺序排列镜头
- 不考虑 timing、转场、音频
- 完整观看一次

第 2 步：识别问题
- 视觉逻辑断裂处
- 节奏问题（拖沓/太快）
- 覆盖缺口（需要补充的镜头）

第 3 步：补充拍摄
- 列出需要的补拍镜头
- 预留 10-15 次生成预算
- 生成并替换

第 4 步：精细剪辑
- 调整 timing
- 添加转场
- 色彩统一
- 音频同步
```

## 🎬 第十一部分：高级技巧

### 提示词优化技巧

**技巧 1: 从动作中间开始**
```
❌ "一个人开始走路"
✅ "一个人正在走路中"

AI 视频模型继续动作比从静止开始更自然
```

**技巧 2: 指定不想要什么**
```
在提示词末尾添加：
"无文字叠加，无水印，无镜头畸变，无突然相机抖动"
```

**技巧 3: 使用电影术语**
```
这些词在训练数据中，可靠产生效果：
- "跟踪镜头"
- "变焦"
- "手持拍摄"
- "焦点转换"
- "主角镜头"
```

**技巧 4: 限制特写的运动复杂度**
```
特写 + 大幅运动 = 难生成
特写 + 静态/微动 = 易生成

广角/中景 + 复杂运动 = 效果好
```

---

### 转场技巧

**AI 友好转场：**
```
1. 匹配剪辑（相似形状/颜色）
2. 运动转场（镜头运动方向一致）
3. 光线转场（明暗变化）
4. 物体转场（前景物体遮挡）

避免：
- 硬切到完全不同场景
- 静态到静态的快速切换
```
